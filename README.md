# How to Install

    git clone <url>
    git submodule init
    git submodule update

Create symlinks to vim, vimrc, gvimrc, and bundle.vim

    ln -nfs /path/to/vimrc ~/.vim
    ln -nfs /path/to/bundle.vim ~/.bundle.vim
    ln -nfs /path/to/vimrc/vimrc ~/.vimrc
    ln -nfs /path/to/vimrc/gvimrc ~/.gvimrc
